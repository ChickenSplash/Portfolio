Application of Colours

When making the Netmatters mirror website, I was presented with the challenge of applying all the different colours on each sibling element, the solution to that was using a SASS mixins to do all the work for me.

So I declare a $theme map, with that, I can iterate through each colour and apply all their colours to all child elements, all I had to do was change a single class on the parent element.

<div class="nav-category theme-gray">
    <a class="nav-button" href="#">
        <span class="icon-cogs theme"></span>
        <span class="small">Bespoke</span>
        <span>Software</span>
        <span class="triangle"></span>
    </a>
    <div class="nav-dropdown">
    </div>
</div>
$theme: (
    "gray": #67809F,
    "yellow": #F7B322,
    "blue": #4183D7,
    "green": #2ECC71,
    "red": #D64541,
    "purple": #926FB1,
    "crimson": #F62459,
);

@mixin nav-bar-theme($theme) {
    @each $name, $color in $theme {
        &-#{$name} {
            .theme {
                color: $color;
            }
            .triangle {
                border-color: $color transparent transparent transparent;
            }
            &:hover {
                background-color: $color;
            }
            &:hover .theme {
                color: #fff;
            }
            .nav-dropdown {
                background-color: color.adjust($color, $lightness: -10%);
            }
        }
    }  
}

.theme {
    @include nav-bar-theme($theme);
}

Drawer Animation Functionality

I first declare an object containing key-value pairs, it represends the button and what the button should open.

Then I make a loop that iterates through each key-value pair and, using jQuery, applied an event listener to the button and upon trggering the event, it opens the corresponding slide.

I then target the close button and add an event listener to make it close its corresponding slide and re-open the original project thumbnails.

const detailsCards = {
    "#netmatters-details": "#netmatters-card",
    "#project-2-details": "#project-2-card",
    "#project-3-details": "#project-3-card",
    "#project-4-details": "#project-4-card",
    "#project-5-details": "#project-5-card",
    "#project-6-details": "#project-6-card",
}

for(const button in detailsCards) {
    $(detailsCards[button]).hide();

    $(button).click(() => {
        $(".projects-work").slideUp(350);
        $(detailsCards[button]).slideDown(350);
    });

    $(detailsCards[button] + " .button.close-slide").click(() => {
        $(detailsCards[button]).slideUp(350);
        $(".projects-work").slideDown(350);
    });
}
